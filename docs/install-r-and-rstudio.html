Install R and RStudio

Install R

  1. Open an internet browser and go to https://cloud.r-project.org/.
  2. Under Download and Install R, click on the link that corresponds to your operating system (e.g., Download R for macOS or Download R for Windows).
  3. Download the latest release by saving the executable file on your computer. Run the .exe file and follow the installation instructions.

Now that R is installed, you need to download and install RStudio.

Install RStudio

  1. Go to www.rstudio.com and click on the “Download RStudio” button.
  2. Click on “Download RStudio Desktop.”
  3. Click on the version recommended for your system, or the latest Windows version, and save the executable file. Run the .exe file and follow the installation instructions
